How do I...?
Please, check the sample projects, they show how to do different things. They are installed by the installer, and they can be found at [Program Files]\BSPrinter\Samples -On Windows 64 bits it is [Program Files (x86)]-.
Or you can access them from the program group that the installer creates.
Alternatively, you can download these sample projects from another file on the download page.

Also, in the reference, each property, method and event is explained.

For general help about VB6 programming, including questions regarding how to use the Printer object, a good place to get help is this forum.
