ToolbarIconsSize Property

Applies to: PrintPreview, IPrinterEx object

Type: BSToolbarIconsSizeConstants


Description:

Returns or sets a value that determines the size of the toolbar icons at the Print Preview dialog box.

The default setting is bsPPTIconsAutoAndUserCanChange.
If it is set to any of the settings that end in "UserCanChange", the user can change the size of the toolbar icons by selecting an option of a menu that appears right clicking on the top toolbar or by clicking the menu button, and this setting will be remembered.
