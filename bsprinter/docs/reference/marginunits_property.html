MarginUnits Property

Applies to: PrintPreview, IPrinterEx object

Type: BSMarginUnitsConstants


Description:

Returns or sets a value that determines the units, Millimeters or Inches, that the program (not for the final user, for the final user see MarginUnitsForUser) will use for the property settings related to margins, which are: LeftMargin, TopMargin, RightMargin, BottomMargin, MinLeftMargin, MinTopMargin, MinRightMargin, MinBottomMargin, LeftNonPrintableArea, TopNonPrintableArea, RightNonPrintableArea, BottomNonPrintableArea, TopMarginEffective and BottomMarginEffective.

The default setting is vbMillimeters.
