PageSetupButtonVisible Property

Applies to: PrintPreview, IPrinterEx object

Type: Boolean


Description:

Returns or sets a value that determines whether a button to configure Page Setup will be available on the top toolbar of the Print Preview dialog box.

The default setting is True.
