AllowUserChangeOrientation Property

Applies to: PrintPreview, IPrinterEx object

Type: Boolean


Description:

Returns or sets a value that determines whether the user will be able to change the page Orientation from the Print Preview and Page Setup dialog boxes. He will still be able to change it from the Printer dialog box. For not displaying the Printer dialog box, use the AllowUserChangePrinter property (he won't be able to choose the printer device either).

The default setting is True.
