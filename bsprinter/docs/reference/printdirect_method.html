PrintDirect Method

Applies to: PrintPreview, IPrinterEx object

  Return Type:     None.  
 
  No parameters.  


Description:

Sends the report directly to the printer, without displaying the Print Preview or Printer dialog boxes.

Note:

If RememberUserPreferences is set to other than bsSPNever, and the user settings have been saved, it will be printed with the last user selected settings.
