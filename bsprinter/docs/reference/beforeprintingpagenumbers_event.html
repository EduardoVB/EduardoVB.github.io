BeforePrintingPageNumbers Event

Applies to: PrintPreview, PrinterExEvents object

  Parameter(s):  
   Returns value     Cancel     As     Boolean  


Description:

Occurs when the page numbers are about to be printed.
The page numbers printing can be canceled at this time by setting the Cancel argument to True.
