PageView Property

Applies to: PrintPreview, IPrinterEx object

Type: BSPageViewConstants


Description:

Returns or sets a value that determines the page view layout that will be selected by default in the Print Preview dialog box.

The default setting is bsPVFitHeight.

Note: in case that this property is set to bsPVNormalSize and the property PageViewNormalSizeButtonVisible is set to False, the 'View normal size' button will be displayed anyway at the toolbar.
