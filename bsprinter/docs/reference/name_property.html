Name Property

Applies to: GridReportStyle object

Type: String


Description:

Returns or sets the name of the Style.

Note:

The name is localized (translated into different languages) and can be customized through the GetLocalizedText event.

See their TextID's in BSUserInterfaceTextIDConstants, they start with bsPPUI_General_GridReportStyleNames_*
