TopMarginEffective Property

Applies to: PrintPreview, IPrinterEx object

Type: Single (Read only)


Description:

Returns a value in millimeters or inches, depending on the setting of the MarginUnits property, which indicates the top margin actually applied. This value may be greater than the one set in the TopMargin property, since the margin space can be increased if necessary to print the page numbers or header.
