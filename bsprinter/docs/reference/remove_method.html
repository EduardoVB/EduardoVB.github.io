Remove Method

Applies to: GridReportStyles object

  Return Type:     None.  
 
  Parameter(s):  
   In     Required     IndexNameOrObject     As     Variant  


Description:

Removes a style from the collection.

The IndexNameOrObject argument must provide a String with the name of the style to remove or a number with its index (position) in the collection.
