StylesAreEqual Method

Applies to: GridReportStyles object

  Return Type:     Boolean  
 
  Parameter(s):  
   In     Required     GridReportStyle1     As     GridReportStyle  
   In     Required     GridReportStyle2     As     GridReportStyle  


Description:

Returns True if the two styles supplied by the GridReportStyle1 and GridReportStyle2 parameters are equal.
