LeftNonPrintableArea Property

Applies to: PrintPreview, IPrinterEx object

Type: Single (Read only)


Description:

Returns a value in millimeters or inches, depending of the setting of the MarginUnits property, which indicates the left physical space where the Printer device cannot print. The LeftMargin cannot be set smaller than that.
