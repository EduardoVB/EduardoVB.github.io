PageNumbersColor Property

Applies to: PrintPreview, IPrinterEx object

Type: Long


Description:

Returns or sets the color that will be used to print the page numbers.
This font color setting can also be changed by the final user from the Page Numbers Options or Format dialog boxes if AllowUserChangePageNumbers is set to True (it is the default setting).

The default color is 0 - vbBlack.
