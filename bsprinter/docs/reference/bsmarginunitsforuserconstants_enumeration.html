BSMarginUnitsForUserConstants Enumeration

Constants used by MarginUnitsForUser property,


bsMUInches = 5 ' The user will see the margin units in inches.
bsMUMillimeters = 6 ' The user will see the margin units in millimeters.
bsMUUserLocale = 0 ' The user will see the margin units in inches or millimeters, depending on Windows locale settings.
