MarginUnitsForUser Property

Applies to: PrintPreview, IPrinterEx object

Type: BSMarginUnitsForUserConstants


Description:

Returns or sets a value that determines the units, Millimeters or Inches, that will be used for the margins in the Printer dialog box.

The default setting is bsMUUserLocale, that means that it will use Millimeters or Inches based in the Windows locale user setting.
