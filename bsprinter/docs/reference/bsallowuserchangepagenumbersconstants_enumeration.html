BSAllowUserChangePageNumbersConstants Enumeration

Constants to set which attributes the user will be able to customize or not.


bsCPAllowChangeAll = 15 ' This constant is the sum of the other flags: bsCPAllowChangeFont OR bsCPAllowChangePosition OR bsCPAllowChangeFormat OR bsCPAllowChangeColor. The user will be able to change all: font, position, format and color.
bsCPAllowChangeColor = 8 ' Determines whether the user will be able to change the color of the page numbers. For this flag to work, bsCPAllowChangeFont flag must also be set.
bsCPAllowChangeFont = 1 ' Determines whether the user will be able to change the font of the page numbers.
bsCPAllowChangeFormat = 4 ' Determines whether the user will be able to change the format of the page numbers.
bsCPAllowChangeNone = 0 ' The user won't be able to change any: font, position, format or color.
bsCPAllowChangePosition = 2 ' Determines whether the user will be able to change the position of the page numbers.
