BSToolbarIconsSizeConstants Enumeration

Constants used by the ToolbarIconsSize property


bsPPTIconsAuto = 4 ' The size of the icons is selected automatically (small, medium or big) according to the screen size.
bsPPTIconsAutoAndUserCanChange = 0 ' The size of the icons is selected automatically (small, medium or big) according to the screen size, and the user has the option to change it by selecting a menu option. The menu is displayed by right clicking on the top toolbar or by clicking the Menu button (see MenuButtonVisible property).
bsPPTIconsBig = 7 ' The size of the icons is big, and the user cannot change it.
bsPPTIconsBigAndUserCanChange = 3 ' The default size of the icons is big, and the user has the option to change it by selecting a menu option. The menu is displayed by right clicking on the top toolbar or by clicking the Menu button (see MenuButtonVisible property).
bsPPTIconsMedium = 6 ' The size of the icons is medium, and the user cannot change it.
bsPPTIconsMediumAndUserCanChange = 2 ' The default size of the icons is medium, and the user has the option to change it by selecting a menu option. The menu is displayed by right clicking on the top toolbar or by clicking the Menu button (see MenuButtonVisible property).
bsPPTIconsSmall = 5 ' The size of the icons is small, and the user cannot change it.
bsPPTIconsSmallAndUserCanChange = 1 ' The default size of the icons is small, and the user has the option to change it by selecting a menu option. The menu is displayed by right clicking on the top toolbar or by clicking the Menu button (see MenuButtonVisible property).
