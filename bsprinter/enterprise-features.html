Enterprise only features:



  • Ability to customize the dialogs button style and font name:

    • The control's properties are: UIButtonStyle and UIFontName.
    This feature is shown in sample: 21 - Customize User Interface look.

  • Ability to customize the Print Preview dialog icons:

    • The control's property is: ToolbarButtonPicture.
    This feature is used in sample: 22 - Save PDF from a toolbar button.

  • Unlimited number of custom grid styles:

    • The number of custom grid report styles that can be added with a non-enterprise license is limited to 2.
    This feature is shown in sample: 20 - Change the default grid style when printing a grid.
