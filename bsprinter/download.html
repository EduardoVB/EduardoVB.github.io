Note: In the current version there is no need to enter a license information.

Minimum supported OS: Windows XP.

If you download and install the component from the installer, it will automatically register the OCX on Windows.
If you download the OCX file without the installer, you'll need to run the IDE elevated (with admin privileges) and manually register it from the menu Project, Components.
